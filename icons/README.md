# Icons

All icons from the cercol.team visual identity, extracted as standalone SVG files.
Monocolor, stroke-based, hand-drawn style. All use `currentColor` so color is
controlled by the parent element.

No external icon libraries — everything is custom.

---

## Moon phase icons — `icons/moon/`

Instrument phase indicators. ViewBox `0 0 32 32`, stroke-based, `fill="none"`.

| File | Description | Usage sizes |
|---|---|---|
| `new-moon.svg` | Dark circle with diagonal hatching (unlit face) | 13px (labels), 18px (results), 36-40px (pages), 80px (home) |
| `first-quarter.svg` | Right half illuminated, inward terminator curve | 13px (labels), 18px (results), 36-40px (pages), 80px (home) |
| `full-moon.svg` | Full disk with three irregular crater marks | 13px (labels, witness), 18px (results), 36px (pages), 80px (home) |
| `last-quarter.svg` | Left half illuminated, mirrored terminator curve | 20px (header), 80px (home) |

## React source — `icons/react/`

`MoonIcons.jsx` is the original React component file from cercol.team containing
all icons as JSX components plus the `RoleIcon` and `DimensionIcon` wrapper
functions. Kept as reference for React projects that import icons as components
rather than standalone SVG files.
