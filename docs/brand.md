# Brand identity

## Brand concept

The Cercol mark is built from two elements: a **crescent moon** and **rice grains**.

The crescent represents cycles, phases, and growth — the core metaphor of the
personality assessment instruments (New Moon, First Quarter, Full Moon, Last Quarter).
The rice grains are a nod to the Valencian landscape (arrossars) and appear
throughout the visual system as decorative details around icons and illustrations.

Together, moon + rice = Cercol.

## Typography note

The original corporate identity manual specifies **Berlingske Serif** (Regular and
Heavy) as the display typeface for titles and brand applications. This is a
commercial font by Playtype.

For web and digital projects, `--mm-font-display` uses **Playfair Display** as
a free substitute. Both are high-contrast editorial serifs with similar proportions.
If a project has a Berlingske Serif license, it should be used instead.

Roboto is the communication typeface (body, UI, web) in both the manual and
mm-design. This is not a substitute — it is the official choice.

## Color combination rules

The brand palette has four colors: red (primary), blue, yellow, and green.
These colors must **never be placed directly on top of each other**. Two brand
colors side by side or overlapping produces visual noise and breaks the identity.

### Correct usage

- **Brand color + white**: the default combination for all contexts. Text, icons,
  and graphics in the brand color sit on a white background (or vice versa).
- **Brand color + black**: used when yellow is the background, since white text
  on yellow has poor contrast. Black text/logo on yellow is the approved version.
- **White on brand color**: the logo and UI elements (nav, headers) appear in
  white when placed on a brand-color background. This is the preferred approach
  for full-bleed sections.
- **Single-color tints**: a brand color can be used at reduced opacity for subtle
  backgrounds (e.g. hover states, status pills), but always paired with white or
  the same color at full strength — never with a second brand color.

### Incorrect usage

- Red text on blue background (or any two-brand-color combination)
- Blue border on green card
- Logo in red on yellow background
- Gradient between two brand colors
- Multicolor logo (the logo is always one color: red, black, or white)

### Summary table

| Background | Text / foreground | Status |
|---|---|---|
| White | Red, Blue, Green, Yellow, Black | OK |
| Red | White | OK |
| Blue | White | OK |
| Yellow | Black | OK |
| Green | White | OK |
| Red | Blue, Yellow, Green | Never |
| Blue | Red, Yellow, Green | Never |
| Any brand color | Another brand color | Never |

## Illustration style

The brand uses a **pointillist illustration system** composed of rice-grain shapes.
Scenes (landscapes, cities, figures) are built entirely from small oval dots in the
brand palette. In mm-design, this style survives in the **animal role icons**, where
rice grains appear as decorative details scattered around each animal silhouette.

Full-scene rice-grain illustrations are currently out of scope for mm-design and
are produced manually per project.
