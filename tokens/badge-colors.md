# Badge Colors — mm-design palette for shields.io

All README badges across mm-design projects must use these hex values.
shields.io accepts 6-digit hex without `#`.

| Color   | Hex      | Usage                              |
|---------|----------|------------------------------------|
| Red     | `cf3339` | Primary accent, errors, alerts     |
| Blue    | `0047ba` | Links, info, secondary accent      |
| Yellow  | `f1c22f` | Warnings, highlights               |
| Green   | `427c42` | Success, data, environment         |
| Black   | `111111` | Neutral dark, backgrounds          |
| White   | `ffffff` | Text on dark badges                |

## Example usage

```markdown
![Status](https://img.shields.io/badge/status-active-427c42?style=flat-square)
![Version](https://img.shields.io/badge/v1.0-0047ba?style=flat-square)
![License](https://img.shields.io/badge/license-MIT-cf3339?style=flat-square)
```

## Badge style

Use `flat-square` consistently across all READMEs.
